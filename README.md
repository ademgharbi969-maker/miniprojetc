# Resau Project

## Description
C program that processes data and creates output files.

## Files
- `resau.c` - Main C source code
- Output files (ignored by Git):
  - `resau.exe` - Compiled executable
  - `relations.txt` - Text output
  - `utilisateurs.bin` - Binary output

## Compilation
```bash
gcc -o resau resau.c
```

## Execution
```bash
./resau
```
